Queensland's Geological Features API

This is an API that delivers information about specific geological (sometimes "geologic") features within the Australian state of Queensland. The data is sourced from the Qld GeoFeatures Dataset. That data is formulated according to the GeoFeatures Ontology which is a profile of the Location Index (LocI) project's ontology. This means Qld GeoFeatures data is conformant with the LocI project requirements.
