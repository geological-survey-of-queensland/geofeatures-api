GSQ's Features of Interest Linked Data Product

This is an API that delivers GSQ's Structural Framework data holdings online as Linked Data. It's called a Features of Interest product since, in the future, it will deliver more than just Structural Framework Data.

This tool uses the SWEET Geological Features Profile Ontology as its main data model and that model conforms to the Location Index (LocI) project's requirements so this information is compatible with other spatial data published by LocI.

Use

Try the links above to brows the Provinces stored in this dataset by type (Craton, Orogen etc.) or list Provinces by age.

Code

The code for this application is online in a Git code repository at https://github.com/geological-survey-of-queensland/gsq-foi-api.
