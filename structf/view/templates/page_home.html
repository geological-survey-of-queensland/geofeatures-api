GSQ's Structural Framework Linked Data Product

This is an API that delivers GSQ's Structural Framework data holdings online as Linked Data.

To do this, the tool uses the SWEET Geological Features Profile Ontology as its main data model and the series of GSQ Structural Framework data web services for its information.
